This project was developed during the Open Data Day Bilbao 2014 event, part of the bigger Open Data Day Hackathon, which tries to gather citizens around the world to write applications, liberate data, create visualizations and publish analyses using open public data.

The event took place at the University of Deusto on the 21st February, 2014. MORElab's team organised the event in order to enjoy a full day playing with Open Data.

Below a list of some software and libraries used during the development of the project can be found:

Python  A programming language to quickly develop complex applications.
Flask  A microframework for Python based on Werkzeug, Jinja 2 and good intentions. "Web development, one drop at a time".
SQLAlchemy  A Python SQL toolkit and ORM that gives application developers the full power and flexibility of SQL.
Beautiful Soup 4  A Python library designed for quick turnaround projects like screen-scraping.
d3js  A JavaScript library for manipulating documents based on data.
sigma.js  A JavaScript library dedicated to graph drawing.
Google Charts  Powerful, simple to use and free chart tools.
JqCloud  A jQuery plugin to draw neat word clouds that actually look like clouds.
networkx  A Python software package for the creation, manipulation and study of the structure, dynamics and functions of complex networks.
Ghephi  An interactive visualization and exploration platform for all kinds of networks and complex systems, dynamic and hierarchical graphs.
genderize.io  An API service which provides probabilistic guess on the gender of a first name, along with a certainty factor.
Bootstrap  The most popular front-end framework for developing responsive, mobile first projects on the web.
Font Awesome  The iconic font designed for Bootstrap
