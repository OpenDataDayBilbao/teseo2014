This project has two main objectives:

  • Make the information kept at teseo available as Open Data.
  • Perform different analyses over the previous data to understand them.

Teseo is a Doctoral Dissertation database maintained by the Spanish Ministry of Education, keeping records for the dissertations defended in Spain since 1977. Teseo's website provides different forms to query the database, but makes it difficult for automatic processing.

Some of performed work involves:

  • Gender dissambiguation for authors and panel members. The process is automatized, so some names could be wrongly assigned a gender.
  • Dissertations are tagged with different topics, from UNESCO's 6-digit nomenclature for fields of science and technology. Topic distribution for different levels of granularity is available.
  • People with the same full name, or with a high probability of being the same person (acronyms, abbreviations, etc.) are assigned to be the same person entity.
  • Analyses are visualized through interactive charts to make them understandable by any user. The  Info button on the top of each chart provides an extended explanation of the rendered data.

All the source code of this website is openly available in the project's GitHub repository.
